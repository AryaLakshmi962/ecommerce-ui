# E-Commerce UI (React)

This is a modern e-commerce frontend built using React.  
The project demonstrates a complete shopping flow including product browsing, cart, checkout, and inventory management.

---

## Features

- Product listing page
- Add to cart functionality
- Cart summary and pricing
- Discount system (NEWYEAR, SAVE20, FLAT50)
- Multi-step checkout process
- Inventory management dashboard
- Stock tracking system
- Clean and modern UI design

---

## Pages

### Shop
Users can browse products and add them to the cart.

### Cart
Users can view selected items, apply discount codes, and see the total price.

### Checkout
Multi-step process:
- Product selection
- Discount selection
- Review
- Payment success

### Inventory
Displays all products with stock levels and total value.

---

## Tech Stack

- React (Vite)
- JavaScript
- CSS
