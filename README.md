# E-Commerce UI (React)

This is a modern e-commerce frontend built using React and Vite.  
The application demonstrates a complete shopping flow including product browsing, cart management, checkout process, and inventory tracking.

---

## Features

- Product listing interface
- Add to cart functionality
- Cart summary with pricing
- Discount system (NEWYEAR, SAVE20, FLAT50)
- Multi-step checkout flow
- Inventory management dashboard
- Stock tracking (In Stock / Low Stock)
- Clean and modern UI design
